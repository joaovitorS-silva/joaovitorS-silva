## Sobre

Tenho 17 anos e sou estudante do curso técnico de **Informática para Internet** no IFRN Campus Caicó. Meu foco principal é o **desenvolvimento backend**, com ênfase em construção de APIs REST, modelagem de banco de dados e boas práticas de autenticação e segurança.

Gosto de escrever código organizado, entender como as coisas funcionam por baixo dos panos e transformar ideias em projetos reais. Atualmente estou evoluindo também no frontend com React.

### 🔗 [PROJETO-COM-FAST_API](https://github.com/joaovitorS-silva/PROJETO-COM-FAST_API)
API REST para gerenciamento de pedidos estilo delivery. Desenvolvida com autenticação via JWT, controle de permissões entre usuários e administradores, modelagem com SQLAlchemy e migrações com Alembic.  
`Python` `FastAPI` `SQLAlchemy` `Alembic` `MySQL` `JWT`

`Python` `FastAPI` `MySQL`
